# 🎵 Melody Masters

## 🌟 Project Overview

**Melody Masters** is a web-based e-commerce platform for buying musical instruments and digital music resources.  
The system includes product management, user authentication, order management, reviews, and admin/staff control.

This project is developed using **PHP, MySQL, HTML, CSS, and JavaScript** and can be deployed locally using XAMPP.

---

## 🚀 Features

### 👤 User Features
- User registration & login
- Browse musical instruments
- View product details
- Add products to cart
- Checkout & order management
- View order history
- Submit product reviews
- Download digital music assets

### 🔐 Admin Features
- Admin dashboard
- Manage products (Add / Edit / Delete)
- Manage customers
- Manage orders
- Manage staff
- Manage reviews
- Update stock

### 👨‍💼 Staff Features
- View assigned dashboard
- Manage product updates
- Update stock information
